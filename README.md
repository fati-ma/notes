# Notes App

## Author: Fatima Atieh


### Overview

A terminal based (CLI) application allowing users to easily create and manage a list of categorized notes

Use the following to run the cli app:

`node index.js --add 'note'`

`node index.js -a 'note'`



## Modules

`inputs.js`

`notes.js`


## Setup

```
npm i minimist
npm  i jest
```
